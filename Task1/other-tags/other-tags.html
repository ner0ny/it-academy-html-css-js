Популярные теги

dl,dt,dd

Тег <dl> входит в тройку элементов <dl>, <dt>, <dd>, предназначенных для создания списка определений. Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина.

Пример

tag
элемент языка разметки гипертекста.Текст, содержащийся между начальным и конечным тегом, отображается и размещается в соответствии со свойствами, указанными в начальном теге.

pre

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один. Тег <pre> позволяет обойти эту особенность и отображать текст как требуется разработчику.

Пример

        -----           |-----
       ||----|          ----||  
       ||-----         -----||  
       ||-----|       |-----||
       || -----       ------||
       ||- ----|     |------||
       ||---||--     -------||
       ||--|| --|   |-------||
       || -|| |--   --- - --||
       || -||  --|-|--| - ---|
       |---||  |-----| |-----|
       |---||   |----  |-----| 
       |----|    ---   |-----|
       |-----          ------|

code

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.

Пример

function checkParent (src, dest) {
while (src != null) {
if (src.tagName == dest) return src
src = src.parentElement
}
return null
}

tt

Контейнер <tt> отображает текст моноширинным текстом. Этот тег относится к группе тегов физического форматирования.

Пример

Lorem ipsum dolor sit amet consectetur adipisicing elit. Provident eius excepturi, nostrum nisi deleniti nesciunt corrupti animi, at, earum illum voluptas distinctio. Esse veritatis, quidem aut delectus dicta error aliquam.

kbd

Тег <kbd> используется для обозначения текста, который набирается на клавиатуре или для названия клавиш. Браузеры обычно помечают текст в контейнере <kbd> моноширинным шрифтом.

Пример

<Alt>+<Delete>

sup и sub

Отображает шрифт в виде верхнего и нижнего индекса.

small

Тег <small> уменьшает размер шрифта на единицу по сравнению с обычным текстом

Пример

Уменьшенный шрифт

big

Увеличенный шрифт

blockquote

Тег <blockquote> предназначен для выделения длинных цитат внутри документа.

Пример

Lorem ipsum dolor sit amet, consectetuer adipiscing elit, sed diem nonummy nibh euismod tincidunt ut lacreet dolore magna aliguam erat volutpat. Ut wisis enim ad minim veniam, quis nostrud exerci tution ullamcorper suscipit lobortis nisl ut aliquip ex ea commodo consequat.

q

Тег <q> используется для выделения в тексте цитат. Содержимое контейнера автоматически отображается в браузере в кавычках.

Пример

Станислав Лец утверждал: Чаще всего выход там, где был вход.

ins

Тег <ins> предназначен для выделения текста, который был добавлен в новую версию документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <ins> как подчеркнутый.

Пример

Вечером был красивый туман, а особенно красиво выглядело солнце

del

Тег <del> используется для выделения текста, который был удален в новой версии документа. Браузеры обычно помечают текст в контейнере <del как перечеркнутый.

Пример

Вечером был красивый туман, а особенно красиво выглядело солнце

Пример

CGI обозначается протокол, с помощью которого любые внешние программы взаимодействуют с веб-сервером. С помощью CGI на сервере можно выполнять программы на любом языке программирования и результат их действия выводить в виде веб-страницы.

address

Тег <address> предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д. По умолчанию текст внутри контейнера <address> отображается курсивным начертанием

Пример

Vladimir Gribovskiy

wbr

Тег <wbr> указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента.

Пример

метоксихлордиэтиламинометилбутил
